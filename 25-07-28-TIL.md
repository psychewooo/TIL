#### 자율 학습
내장 함수를 이용하지 않고 최소값, 최대값 구하기
```python
def find_min_max(input_list):

    # 주어진 리스트를 오름차순으로 정렬
    input_list.sort()

    # 정렬 시에 첫 번째 값은 최솟값이 되고 마지막 값은 최댓값이 된다
    return input_list[0], input_list.pop()


result = find_min_max([3, 1, 7, 2, 5])
print(result)
```
